# Integer

### signed vs unsigned  
signed : positive, negative
use one bit to clasify the negative(1) or positive(0) 
=> (MSB) Most Significant Bit (최상위 비트)

unsigned : only positive

ex) char(2^8=256)

signed char ( -128 ~ +127 ) 
=> if positive = MSB(최상위 비트)is 0
   if negative = MSB(최상위 비트)is 1
   
unsigned char( 0 ~ +255 )
